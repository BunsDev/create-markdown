# Changelog

All notable changes to create-markdown are documented here.

## [0.2.2] - Latest

### Changed

- Maintenance release
- Updated dependencies

## [0.2.1]

### Added

- Live inline rendering with markdown shortcuts
- Keyboard shortcuts (Cmd+B, Cmd+I, Cmd+U)
- List item management with Enter/Backspace/Tab

### Fixed

- Block selection improvements
- Content synchronization issues

## [0.2.0]

### Added

- Complete block-based architecture
- Block factory functions (h1, paragraph, etc.)
- Document operations (insert, remove, update, move)
- React hooks (useDocument, useMarkdown, useBlockEditor)
- BlockRenderer component

### Changed

- Rewrote parser for better accuracy
- Improved TypeScript types

## [0.1.0]

### Added

- Initial release
- Basic markdown parsing
- Basic serialization
- TypeScript support
