# Roadmap

This roadmap outlines the planned development of create-markdown.

## Current Status

The project is actively maintained with the following packages available:

| Package | Status | Version |
|---------|--------|---------|
| @create-markdown/core | Stable | 0.1.x |
| @create-markdown/preview | Stable | 0.1.x |
| @create-markdown/react | Stable | 0.1.x |
| @create-markdown/mdx | New | 0.1.x |
| create-markdown | Stable | 0.2.x |

## Planned Features

### Near Term

- **Nested lists** - Support for nested bullet and numbered lists
- **Custom block types** - Plugin system for custom blocks
- **Collaborative editing** - Real-time collaboration support
- **Better table editing** - Visual table editor component

### Medium Term

- **Vue components** - @create-markdown/vue package
- **Svelte components** - @create-markdown/svelte package
- **Solid components** - @create-markdown/solid package
- **Math support** - KaTeX integration for math rendering

### Long Term

- **CRDT support** - Conflict-free replicated data types
- **LSP integration** - Language server for editor support
- **VS Code extension** - Native VS Code integration
- **Mobile support** - React Native components

## Version Policy

We follow [Semantic Versioning](https://semver.org/):

- **Major** (1.0.0): Breaking changes
- **Minor** (0.1.0): New features, backwards compatible
- **Patch** (0.0.1): Bug fixes
